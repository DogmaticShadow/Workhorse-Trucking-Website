# Workhorse Trucking Website

## Setup Instructions

1. Save the truck image (showing two Workhorse trucks) as `trucks-foreground.jpg`
2. Save the rocky quarry image as `quarry-background.jpg` 
3. Open `index.html` in your browser

## Images Needed

- **trucks-foreground.jpg**: The image showing the white and red Workhorse trucks at the construction site
- **quarry-background.jpg**: The rocky quarry/mining site image (will be blurred as background)

## Features

- Responsive design optimized for trucking company
- Hero section with truck imagery
- Call-to-action buttons positioned between the trucks
- Professional navigation
- Mobile-friendly layout

The website is designed to showcase Workhorse Trucking's services with the actual truck fleet as the focal point.
